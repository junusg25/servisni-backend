# Servisni Backend

This is the backend for the **Servisni Modul** application. It provides authentication, machine tracking, repairs management, and part inventory.

## Features
- User Authentication (JWT)
- Machines and Parts Management
- Repairs Tracking
- PostgreSQL Database
